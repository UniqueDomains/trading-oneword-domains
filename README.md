## 🧱 Field summary

- `domain` — Fully qualified domain name.
- `status` — Current acquisition state for the domain in the public extract.
- `purchase_price` — Visible purchase price when available.
- `renewal_price` — Visible renewal price when available.
- `attractiveness` — Composite naming score used as a decision-support signal.
- `demand` — Relative buyer-pressure score when available.
- `length` — Character count without the TLD.
- `registrar` — Registrar name when known.
- `created_at` — Creation timestamp when known.
- `expires_at` — Expiry timestamp when known.

## ⚠️ Methodology and caveats

This selection is defined by a single trait: every name ends in .trading. The set is broad, with personal names, generic words, category terms, and potentially sensitive or trademark-loaded strings appearing side by side. Examples such as Liam.trading, Laura.trading, facts.trading, and girls.trading show how varied the naming styles are. When comparing these domains, focus first on whether the word becomes clearer or weaker when paired with .trading. Then check asking price against long-term renewal tolerance, and rule out names that could create trademark or reputation friction, such as obvious brand references.

- All names in this set use the .trading extension
- Median ask is 59.36 across 9,819 domains
- Word quality varies from generic terms to personal names
- Screen for trademark and reputation risk before buying
